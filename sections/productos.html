El monitor de pantalla táctil Elo 1302L está diseñado con el mismo estilo que abarca toda la línea de monitores de la serie 02. Con un diseño compacto para una fácil integración, el 1302L ofrece la flexibilidad para instalarlo como un terminal de punto de venta tradicional o un punto de interacción orientado al cliente, lo que lo hace ideal para el punto de venta, el autoservicio, la señalización y aplicaciones de hotelería. Proporciona una interfaz USB tipo C que proporciona toque, video y energía. Esto reduce la cantidad de cables necesarios para cualquier configuración de 3 a 1 y hace que la instalación y el servicio sean muy fáciles. En caso de que su PC aún no esté equipada con USB Type-C, puede usar el HDMI o VGA tradicional para conectarse a su PC.

El monitor táctil LED panorámico de 15.6 pulgadas 1509L de Elo proporciona una solución reforzada para uso minorista y de bajo perfil con una construcción de cristal puro para permitir una alta transmisión de luz sin comprometer la durabilidad o funcionalidad. Disponible con IntelliTouch de ondas acústicas de superficie (SAW) que ofrece capacidad táctil simple, el 1509 ha sido construido para ser táctil, ofreciendo una solución ideal para aplicaciones generales que incluyen hospitalidad y entornos minoristas.

El monitor táctil de formato estándar 1517L de Elo está construido para tolerar las exigencias del uso público continuo con un diseño resistente creado para ser táctil. Su aspecto moderno, delgado y elegante, combinado con un conjunto de características enfocadas en el mercado minorista, es una solución atractiva para las necesidades de monitores táctiles comerciales.
